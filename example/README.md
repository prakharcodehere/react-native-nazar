# react-native-nazar Example

A demo app showcasing all features of `react-native-nazar`.

## Running the example

```bash
cd example
npm install

# iOS
npx pod-install
npm run ios

# Android
npm run android
```

## Features Demonstrated

- **PerformanceProvider** — Context-based performance data
- **Device Specs** — CPU, memory, storage, battery, thermal info
- **Performance Scoring** — Overall score with CPU/Memory/Storage/GPU breakdown
- **Adaptive Quality** — Automatic settings based on device tier
- **Conditional Rendering** — Feature flags for animations, blur, parallax, etc.
- **Full Benchmark** — Run comprehensive device benchmarks
- **DevTools Overlay** — Floating debug panel
